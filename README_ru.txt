Инструкции по установке.
Вариант без установки питона (простой, но может подтормаживать)
1. Скачать последнюю версию с https://sourceforge.net/projects/panurg-wiz/files/
2. Распаковать и радоваться

Вариант с установкой питона (не всякий гуманитарий осилит)
1. Ставим питон: http://bit.ly/HNkyoU
ставить лучше по дефолтному пути
2. Качаем и ставим по очереди модули для питона:
http://bit.ly/UAFQNH
http://bit.ly/XW6Sd1
http://bit.ly/VW9iLa
http://bit.ly/WyXSgl
http://bit.ly/UYAzNy
http://bit.ly/WCTrTv
3. Добавляем в Path "C:\PYTHON27" если ставили питон в директорию по умолчанию. Как это сделать - инструкция с картинками: http://bit.ly/yVzbgE
4. Выставляем совместимость  c:\python27\python.exe с Windows 7 **ДАЖЕ** если уже стоит семерка. Иначе скрипт будет падать. Почему так я не знаю, это автор python-tesseract что-то набыдлокодил, а я это говноподелие использую в своем скрипте. Опять инструкция: http://bit.ly/11qlRUE
5. Качаем http://bit.ly/VW9utL
внутри там есть директория tessdata, распаковываем ее рядом со скриптом.

Вроде ничего не забыл.

Использование:
1. Запускаем wiz
2. Вводим желаемое количество BP
3. Выбираем желаемое сочетание расы/пола
4. Запускаем WO и начинаем создавать персонажа
5. Висим на экране выбора расы
6. Жмем F1 для начала генерации персонажа
7. Ждем :3
8. Работу скрипта можно прервать по Esc
